This is a demonstration of how to use the various services/libraries/frameworks that exist to make an internet enabled button. In this tutorial we're going to build a button that, when pressed, sends a text message with the current forecast.

What we'll be using:

  • Spark Core - https://store.spark.io/?product=spark-core
  • Twilio API (sms/mms) - https://www.twilio.com/api
  • forecast.io API (weather) - https://developer.forecast.io
  • Parse Cloud - https://parse.com

What we will not be doing

This tutorial is not intended as an in depth guide on each of these technologies. I'll include relevant links in situations where there may be questions.

Hardware

This circuit was built based on the schematic on arduino.cc. http://www.arduino.cc/en/tutorial/button
